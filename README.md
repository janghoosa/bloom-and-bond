# Bloom & Bond

MBTI와 애착유형을 함께 분석하는 관계 해석 검사.

**https://bloom-and-bond.pages.dev**

## 기능

- MBTI 4축 + 애착유형(불안/회피) 통합 검사
- 감정코드 기반 결과 공유 및 링크 공유
- 두 사람 비교 분석 (궁합, 마찰 포인트, 대화 팁)
- 공유용 이미지 카드 생성 (Canvas API)
- 서버 미사용 — 모든 데이터는 브라우저에서만 처리

## 기술 스택

- React 19 + Vite 7
- HeroUI v3 (RC)
- Tailwind CSS v4
- GA4 이벤트 트래킹
- Cloudflare Pages 배포

## 시작하기

```bash
pnpm install
pnpm dev
```

## 빌드

```bash
pnpm build
pnpm preview
```
